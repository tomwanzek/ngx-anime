Fade In/Out A basic animation using input parameters.

This basic animation uses animation input parameters to control the opacity and size of an SVG circle, respectively, as well as the animation duration. The animation uses two discrete states in and out. The value setting controls the opacity and also drives the fraction of the circles full radius.

State changes trigger the animation based on the then current value and duration settings.
